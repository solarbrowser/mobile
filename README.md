Running Solar Browser
Make sure you have a device or emulator connected, then run:
  ```
  flutter pub get
  flutter run --release
  ```
or for debug mode:
  ```
  flutter pub get
  flutter run --debug
  ```

Building APK
To build the project for release:
  ```
  flutter build apk
  ```
The generated APK will be located at:
  ```
  build/app/outputs/flutter-apk/app-release.apk
  ```
